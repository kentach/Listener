## 1. サービス概要

### 1-1. アプリ開発の背景
- 本アプリは、前職の英会話スクール専用のアプリになります。
- 英会話スクールに通う生徒が「音声を活用した英語の自宅学習」を気軽に行えるようにすることを目的として開発しました。
- 英検5級〜準1級に対応の自社開発のオリジナル教材を収録しており、生徒はスマホやタブレット一つで、音読トレーニングやリスニングを行うことができます。

### 1-2. アプリ開発の主な機能
- 管理機能（許可した生徒のみログイン可）
- 英語学習アプリに欠かせない音声の再生・リピート・スキップ・速度変換
- テキストの登録機能
- 音声のお気に入り登録機能
- 英検のスコア結果をグラフで表示する機能（Reading/Listening/Speaking/Writing各スコアの推移をグラフで表示）
- 音声に完了・未完了を残す機能

## 2. このアイデアはどこから生まれたか

### 2-1. きっかけとなった体験
### 【課題① 生徒の大半が単語学習のみ。英語に触れる機会が圧倒的に不足している。】
- このアプリを開発するきっかけとなったのは、生徒との個別面談でした。
  面談を通して、生徒大半が授業外での自宅学習が「単語の暗記のみ」になっており、英語を読んだり、聞いたりする習慣が十分に身についていないことに気付きました。

### 【課題② リスニング学習のハードルが高い】
- 生徒との個人面談で、「リスニングが苦手だが、どのように勉強すればよいかわからない」という相談を受けることがよくありました。
  私は生徒には授業で使用している教材の音声を活用した「スクリプト音読学習法」を提案していましたが、音声ファイルはCDとダウンロード形式でしか提供されておらず、学習ツールとして使いづらいという課題がありました。

### 【原因】
- スクールで使用されているオリジナル教材には音声CDやQRコードが付属していましたが、音声ファイルをダウンロードしなければならず、学習したい音声へすぐにアクセスできる環境ではありませんでした。
- 同じ音声を繰り返し聞く際には毎回手動で巻き戻す必要があり、非常に手間がかかります。
- また、スマートフォンが学習ツールとして当たり前になった現代において、このような小さな手間は生徒の学習モチベーションを妨げる要因になり得ます。
- 特に、自宅学習の習慣が十分に身についていない生徒にとっては、音声学習から離れてしまう原因の一つになっていると考えました。

### 【解決策】
- 生徒がスマートフォンやPCから簡単に教材音声へアクセスできる環境を提供し、自宅でも気軽に音読練習やリスニング学習に取り組めるよう、本アプリを開発しました。

### 2-2. なぜそれが「気になった」のか

1. **単語学習だけでは、不十分**
- 単語学習を中心に進めること自体には異論はありません。実際、語彙力は英語力の土台であり、学習初期に重点的に取り組むべき重要な要素です。
  しかし、単語学習だけで英語が思うように上達するわけではありません。
  英語学習者が英語の土台を固めていくには、単語を覚えることに加えて、実際に英語を読む、音読する、聞くといった学習を継続することが不可欠です。
- 私の経験ですが、知識として単語を知っているだけでは、実際の英文や音声の中でその単語を瞬時に理解できるようにはなりません。
  特に英語を日常としない日本人は、単語学習と並行して実際に英語に読む・聞く機会を増やし、「知っている単語」を「使える単語」「理解できる単語」へと変えていくことが重要だと考えています。

2. **生徒に授業を復習するきっかけを作りたい**
- 「2-1. きっかけとなった体験」で述べたように、生徒多くが授業で学習した長文やリスニング教材をその場限りで終わらせてしまい、学習したことを復習する習慣が十分に身についていないのが現状です。
- 初学者が英語の土台を築くためには、一度学習した長文やリスニング教材を「繰り返し」復習することが欠かせません。同じ教材に何度も触れることで、授業で学んだ単語や文法、表現が定着し、英語を読む力や聞く力の向上につながります。
そのため、授業で学習した内容を授業内だけで完結させるのではなく、家庭学習の中でも継続的に復習できる仕組みを整えることが重要であると考えました。

### 5-2. それでも自分が作りたい理由
1. オリジナルテキストとリンクしたオンライン学習コンテンツが存在しない
2. 英語を音読する学習習慣を身につけさせたい
- 英語を理解する土台を形成する上で必要なのは「量 × 時間」です。しかし教室での学習時間だけでは十分な練習量を確保できません。
本アプリを活用して自宅学習の機会を増やすことで、授業内容の定着と英語力の向上が期待できます。
- また、「宿題として課題を出す → 自宅で音読・リスニング練習を行う → 講師が面談等で確認する」という学習サイクルを作り、より多くの生徒に音読を英語学習の一つとして定着させることができます。こうすることで、生徒が自宅以外で日常的に英語に触れる機会を増やすことができます。

### 懸念点①

- 何が問題になりそうか：モチベーションが低い生徒にも使ってもらえるか。
- なぜそう思うか：英語学習においては、もともと学習習慣が定着していない生徒も多く、自由利用型の学習ツールでは継続率が低下する可能性があるため。

### 懸念点②

- 何が問題になりそうか：全校生徒数千人規模で利用した場合、アプリ上で負荷や不具合が発生する可能性がある。
- なぜそう思うか：音声再生や教材アクセスが同時に集中することで、サーバー負荷の増加やレスポンス遅延が起こる可能性があるため。
　また、実運用規模での負荷テストが未実施の場合、想定外のバグが顕在化する可能性があるため。
- そのために考えている対策：負荷分散を考慮したインフラ設計や、段階的リリース（テスト運用→一部クラス→全校展開）を行う。
